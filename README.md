# أكاديمية أشبال الجديدة - النسخة المُصلَحة

## التعديلات والإصلاحات

### 🔒 أمان
1. **حذف `ADMIN_PASSWORD_HASH` الوهمي** من `.env` (كان `$2a$10$YourHashedPasswordHere` غير مُستخدم).
2. **Rate limiting** على `/api/auth/login` (10 محاولات كل 15 دقيقة لكل IP).
3. **CORS** أصبح مقيّداً عبر `CORS_ORIGIN` في `.env`.
4. **فلترة أنواع الملفات** المرفوعة (صور/فيديو فقط).
5. **تحقق من إدخالات التسجيل** (طول كلمة المرور، صيغة الجوال).
6. **منع المدير من حذف نفسه**.

### 🛡️ صلاحيات
- إضافة middleware `requirePermission` يفحص جدول `permissions` فعلياً لكل مدرب (admin يتجاوزها). كان مهملاً سابقاً.

### 🐞 إصلاح أخطاء
- التحقق من `req.file` قبل القراءة في رفع الصور/الفيديو (بدلاً من 500).
- منع قبول طلب تسجيل مرتين (UNIQUE conflict).
- التحقق من نوع `media` (`photos`/`videos`) في endpoint التثبيت.
- التحقق من الحقول المطلوبة في `nominees`, `weekly-stars`, `results`, `votes`.

### 🧹 تنظيف
- حذف الملفات الفعلية من القرص عند حذف صورة/فيديو/مرشح (كانت تتسرب).
- معالج أخطاء `multer` موحّد.

### 🌐 تقديم الواجهة
- إصلاح `app.use(express.static(...))` ليقدّم من `__dirname` (بدل مجلد `public` غير موجود).
- إصلاح catch-all ليستثني `/api` و `/uploads`.

## التشغيل

```bash
npm install
npm run init-db   # اختياري - يُنشأ تلقائياً عند البدء
npm start
```

افتح: http://localhost:3000
